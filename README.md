# La Bodega — Lola case study

Standalone UX portfolio case study for **Lola**, La Bodega’s bilingual WhatsApp assistant. This project is separate from the [Competitor Watch](../competitor-watch) retail app.

## Stack

- React 19 + Vite 6
- Tailwind CSS 4
- Motion (animations)

## Run locally

```powershell
cd "c:\My Web Sites\lola-case-study"
npm install
npm run dev
```

Open **http://localhost:5173/**

## Build & preview

```powershell
npm run build
npm run preview
```
